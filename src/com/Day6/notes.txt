Streams API
	- Design to process large amount of data
	- It works on collection, but it is not a data structure
	- It is designed for lambdas
	- It perfoms basic tasks such as search,sort,filter,reduce,map etc.
	- It has 2 types of operations
		- Intermediate Operations: There can be 0 or more intermediate operations.
		- Terminal Operations: there can be 1 terminal operation, which closes the stream.
